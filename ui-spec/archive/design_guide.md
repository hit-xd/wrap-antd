# 私行设计规范（Design Guide）

> 本文档定义与技术实现无关的私行设计规范，用于指导前端代码生成。  
> 开发人员应结合项目技术栈（如 Ant Design、TailwindCSS 等）实现以下视觉与交互要求。

---

## 一、设计令牌总览

### 1.1 品牌色（金色系）

| 设计项 | 色值 | 使用场景 |
|--------|------|---------|
| 主色（品牌金） | `#C5A267` | 主按钮、链接、选中态、品牌标识 |
| 主色悬停态 | `#D1BB8E` | 主按钮 hover |
| 主色激活态 | `#B0915C` | 主按钮 active / pressed |
| 主色浅底 | `#FFFCF0` | 菜单激活背景、品牌色浅色底纹 |
| 主色边框 | `#DFCFB4` | 按钮 focus 边框、品牌相关边框 |
| 主色深色强调 | `#785A32` | 渐变终点、深色强调元素 |

### 1.2 功能色

| 语义 | 色值 | 使用场景 |
|------|------|---------|
| 成功/正向 | `#00C078` | 成功提示、完成状态、正增长 |
| 成功浅底 | `#EBFAF5` | 成功类标签背景 |
| 警告 | `#FFA000` | 警告提示、待处理、中等风险 |
| 警告浅底 | `#FFFBE6` | 警告类标签背景 |
| 错误/危险 | `#FF4D4F` | 错误提示、删除、驳回、负增长 |
| 错误浅底 | `#FFF1F0` | 危险类标签背景 |
| 信息/链接 | `#1874FF` | 普通链接、信息提示、引导 |
| 信息浅底 | `#E6F7FF` | 信息类标签背景 |

### 1.3 中性色

| 用途 | 色值 | 使用场景 |
|------|------|---------|
| 白色 | `#FFFFFF` | 卡片背景、弹窗背景 |
| 页面背景 | `#E8EAF2` | 页面全局底色 |
| 模块背景 | `#F4F6F8` | 表头、模块底色、失效背景 |
| 标题/重点文字 | `#262626` | 页面标题、卡片标题 |
| 正文/主要文字 | `#595959` | 表格内容、表单标签、正文 |
| 次要文字 | `#8C8C8C` | 辅助说明、时间戳 |
| 禁用文字 | `#BFBFBF` | 禁用状态文字 |
| 通用边框/分割线 | `#D9D9D9` | 表格边框、输入框边框、分割线 |
| Hover 边框 | `#DFCFB4` | 输入框 focus / hover 边框 |
| 深色遮罩 | `rgba(0,0,0,0.3)` | 弹窗、抽屉背景遮罩 |
| 浅色遮罩 | `rgba(255,255,255,0.75)` | Loading 遮罩 |

完整灰阶参考：`#F4F6F8`、`#F1F2F7`、`#E8EAF2`、`#D9D9D9`、`#BFBFBF`、`#8C8C8C`、`#595959`、`#262626`、`#0F1829`。

## 二、字体与排版

### 2.1 字体家族
```
"Helvetica Neue", Helvetica, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", Arial, sans-serif
```
数字专用字体（如有）：`"CMBSans"`

### 2.2 字阶（字号 / 行高 / 字重）

| 层级 | 字号 | 行高 | 字重 | 使用场景 |
|------|------|------|------|---------|
| 超大数字 | 40px | 1.2 | 600 | 关键指标卡片数值 |
| 大数字 | 32px | 1.2 | 600 | 仪表盘 KPI 数值 |
| 中数字 | 28px | 1.2 | 600 | 统计卡片数值 |
| 大标题 | 24px | 1.2 | 600 | 页面主标题 |
| 模块标题 | 20px | 1.5 | 600 | 卡片标题、区块标题 |
| 正文（大） | 16px | 1.5 | 400 | 部分强调正文 |
| 正文（默认） | 14px | 1.5 | 400 | 表格内容、表单字段、描述文字 |
| 表格文字 | 15px | 1.5 | 400 | 表格内容专用（可选） |
| 辅助/说明 | 12px | 1.5 | 400 | 提示文本、时间戳、次要信息 |

> 行高规则：标题使用 `1.2`，正文使用 `1.5`。

### 2.3 字重映射

| 字重 | 数值 | 使用场景 |
|------|------|---------|
| 常规 | 400 | 正文、辅助文字 |
| 中粗 | 500 | 强调正文、按钮文字 |
| 半粗 | 600 | 标题、表头 |
| 粗体 | 700 | 极少使用（特殊强调） |

## 三、间距

基础单位：4px。所有间距使用 4 的倍数。

| Token 名 | 值 | 使用场景 |
|----------|----|---------|
| space-1 | 4px | 图标与文字间距、紧凑内边距 |
| space-2 | 8px | 组件内部元素间距、表单项内边距 |
| space-3 | 12px | 表单项之间垂直间距 |
| space-4 | 16px | 页面内边距、卡片内边距、按钮左右间距 |
| space-5 | 20px | 弹窗 body 内边距、表单行间距 |
| space-6 | 24px | 大区块间距、模态框内容区边缘 |
| space-8 | 32px | 页面区块分隔 |
| space-12 | 48px | 页面大分隔 |

## 四、圆角

| 层级 | 值 | 使用场景 |
|------|----|---------|
| xs | 2px | 极小控件、徽标 |
| sm | 4px | 按钮、输入框、下拉框、标签 |
| md | 6px | 卡片容器 |
| lg | 8px | 弹窗、抽屉 |
| round | 999px | 头像、圆形按钮 |

## 五、阴影

| 层级 | 视觉描述 | 使用场景 |
|------|---------|---------|
| sm | 轻微阴影 | 普通卡片 |
| md | 中等阴影 | 卡片悬停、下拉菜单 |
| lg | 较大阴影 | 模态框 |
| card | 卡片默认 | 卡片容器 |
| hover | 悬停提升 | 卡片 hover |
| header | 顶部导航 | 页面头部 |
| modal | 浮层 | 弹窗、抽屉 |

阴影参数（供开发参考）：
```css
--shadow-sm: 0 1px 4px rgba(0,0,0,0.06);
--shadow-md: 0 6px 16px rgba(0,0,0,0.08), 0 3px 6px -4px rgba(0,0,0,0.12);
--shadow-lg: 0 12px 48px 16px rgba(0,0,0,0.03), 0 9px 28px 8px rgba(0,0,0,0.05), 0 6px 16px -8px rgba(0,0,0,0.08);
```

## 八、动画时长与缓动

| 时长 | 值 | 使用场景 |
|------|----|---------|
| 快速 | 0.1s | 微交互、hover 颜色变化 |
| 正常 | 0.2s | 弹窗展开、折叠面板 |
| 慢速 | 0.3s | 页面切换、较大动效 |

缓动函数：`cubic-bezier(0.34, 0.69, 0.1, 1)`（标准），`linear`（线性）。

## 十、组件视觉规范（技术无关）

### 10.1 按钮

| 类型 | 背景 | 边框 | 文字颜色 | 高度 | 圆角 | 字重 |
|------|------|------|---------|------|------|------|
| 主按钮 | 主色 `#C5A267` | 无 | 白色 | 32px | 4px | 500 |
| 次按钮 | 白色 | `#D9D9D9` | `#595959` | 32px | 4px | 400 |
| 危险按钮 | 错误色 `#FF4D4F` | 无 | 白色 | 32px | 4px | 500 |
| 文字按钮 | 透明 | 无 | 信息色 `#1874FF` | 自动 | 无 | 400 |

- 按钮组合：同一区域最多一个主按钮；弹窗底部按钮顺序为“取消”（次按钮）在左，“确定”（主按钮）在右。
- 尺寸变体：小按钮高度 24px（用于表格行内），默认 32px，不使用 40px 大按钮。

### 10.2 表单

- **标签位置**：默认左上（垂直布局）；字段较多时使用右置标签（水平布局），标签宽度 120px。
- **输入框高度**：32px（默认），24px（紧凑）。
- **边框圆角**：4px。
- **边框颜色**：默认 `#D9D9D9`，hover/focus 时变为 `#DFCFB4`。
- **校验错误**：边框变为错误色 `#FF4D4F`，底部显示 12px 错误提示文字（错误色）。

### 10.3 表格

- **表头背景**：模块背景 `#F4F6F8`
- **表头文字**：半粗（600），正文色 `#595959`
- **行高**：默认 54px（中等密度），可选紧凑 40px
- **单元格内边距**：左右 16px
- **边框**：底部边框，颜色 `#D9D9D9`
- **分页**：右对齐，显示总数、每页条数切换、页码跳转

### 10.4 弹窗（Modal / Drawer）

- **圆角**：8px（lg）
- **头部内边距**：16px 24px，底部边框 `#D9D9D9`
- **内容区内边距**：20px（space-5）
- **底部内边距**：12px 20px，顶部边框 `#D9D9D9`
- **遮罩**：深色遮罩 `rgba(0,0,0,0.3)`
- **按钮**：主按钮在右，次按钮在左

## 十一、交互状态定义

| 状态 | 视觉变化 |
|------|---------|
| Hover | 主按钮背景变为主色悬停态 `#D1BB8E`；次级按钮边框加深或阴影提升 |
| Active / Pressed | 主按钮背景变为主色激活态 `#B0915C` |
| Focus | 添加 2px 边框，颜色为主色边框 `#DFCFB4`，可选外发光 |
| Disabled | 文字透明度降至 40% 或使用禁用文字色 `#BFBFBF`，背景变灰，不可交互 |
| Loading | 显示旋转图标，按钮文字可选隐藏或保留，不可再次点击 |

| 设计意图 | 推荐使用组件库（Ant Design 示例） |
|---------|--------------------------------|
| 主按钮 | `<Button type="primary">` |

| 危险按钮 | `<Button danger>` |
| 文字按钮 | `<Button type="link">` |

| 选择器 | `<Select />` |
| 表格 | `<Table />` |
