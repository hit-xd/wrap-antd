# 私行设计系统规范 v1.0

> 本文档由设计系统 CSS 变量转化而来，定义原子级设计 token。  
> 开发人员应根据此规范结合项目技术栈实现样式。

---

## 一、颜色 (Colors)

### 1.1 品牌色（金色系）

| 设计项 | 色值 | 使用场景 |
|--------|------|---------|
| 主色（品牌金） | `#C5A267` | 主按钮、链接、选中态、品牌标识 |
| 主色悬停态 | `#D1BB8E` | 主按钮 hover |
| 主色激活态 | `#B0915C` | 主按钮 active / pressed |
| 主色浅底 | `#FFFCF0` | 菜单激活背景、品牌色浅色底纹 |
| 主色边框 | `#DFCFB4` | 按钮 focus 边框、品牌相关边框 |
| 主色深色强调 | `#785A32` | 渐变终点、深色强调元素 |

| 警告浅底 | `#FFFBE6` | 警告类标签背景 |
| 错误/危险 | `#FF4D4F` | 错误提示、删除、驳回、负增长 |
| 错误浅底 | `#FFF1F0` | 危险类标签背景 |
| 信息/链接 | `#1874FF` | 普通链接、信息提示、引导 |

### 1.3 中性色

| 设计项 | 色值 | 使用场景 |

## 二、字体与排版 (Typography)

### 2.1 字体家族

默认：`"Helvetica Neue", Helvetica, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", Arial, sans-serif`

### 2.2 字号与行高

| 层级 | 字号 | 行高 | 使用场景 |
|------|------|------|---------|
| 超大数字 | `40px` | `1.2` | 关键指标卡片数值 |
| 大数字 | `32px` | `1.2` | 仪表盘 KPI 数值 |
| 中数字 | `28px` | `1.2` | 统计卡片数值 |
| 大标题 | `24px` | `1.2` | 页面主标题 |
| 模块标题 | `20px` | `1.5` | 卡片标题、区块标题 |
| 正文 | `16px` / `14px` | `1.5` | 表格内容、表单字段、描述文字（私行正文实际用 14px 居多，此处保留 16px 作为变体） |
| 表格文字 | `15px` | `1.5` | 表格内容专用 |
| 辅助/说明 | `12px` | `1.5` | 提示文本、时间戳、次要信息 |

> 行高规则：标题使用 `1.2`，正文使用 `1.5`。

### 2.3 字重

| 字重 | 数值 | 使用场景 |
|------|------|---------|
| 常规 | `400` | 正文、辅助文字 |
| 中粗 | `500` | 强调正文、按钮文字 |
| 半粗 | `600` | 标题、表头 |
| 粗体 | `700` | 极少使用（特殊强调） |

## 三、间距 (Spacing)

基础单位：4px。以下为常用间距 token：

| Token 名（语义） | 值 | 使用场景 |
|----------------|----|---------|
| `space-1` | 4px | 图标与文字间距、紧凑内边距 |
| `space-2` | 8px | 组件内部元素间距、表单项内边距 |
| `space-3` | 12px | 表单项之间垂直间距 |
| `space-4` | 16px | 页面内边距、卡片内边距、按钮左右间距 |
| `space-5` | 20px | 弹窗 body 内边距、表单行间距 |
| `space-6` | 24px | 大区块间距、模态框内容区边缘 |
| `space-8` | 32px | 页面区块分隔 |
| `space-12` | 48px | 页面大分隔 |

## 四、圆角 (Border Radius)

| 层级 | 值 | 使用场景 |
|------|----|---------|
| `xs` | 2px | 极小控件、徽标 |
| `sm` | 4px | 按钮、输入框、下拉框、标签 |
| `md` | 6px | 卡片容器 |
| `lg` | 8px | 弹窗、抽屉 |
| `round` | 999px | 头像、圆形按钮 |

## 五、阴影 (Shadows)

| 层级 | 视觉描述 | 使用场景 |
|------|---------|---------|
| `sm` | 轻微阴影 | 普通卡片 |
| `md` | 中等阴影 | 卡片悬停、下拉菜单 |
| `lg` | 较大阴影 | 模态框 |
| `card` | 卡片默认 | 卡片容器 |
| `hover` | 悬停提升 | 卡片 hover |
| `header` | 顶部导航 | 页面头部 |
| `modal` | 浮层 | 弹窗、抽屉 |

具体阴影参数（供开发参考）：
- `sm`: `0 1px 4px rgba(0,0,0,0.06)`
- `md`: `0 6px 16px rgba(0,0,0,0.08), 0 3px 6px -4px rgba(0,0,0,0.12)`
- `lg`: `0 12px 48px 16px rgba(0,0,0,0.03), 0 9px 28px 8px rgba(0,0,0,0.05), 0 6px 16px -8px rgba(0,0,0,0.08)`

## 八、动画时长与缓动

| 时长 | 值 | 使用场景 |
|------|----|---------|
| 快速 | `0.1s` | 微交互、hover 颜色变化 |
| 正常 | `0.2s` | 弹窗展开、折叠面板 |
| 慢速 | `0.3s` | 页面切换、较大动效 |

缓动函数：`cubic-bezier(0.34, 0.69, 0.1, 1)`（标准曲线），`linear`（线性）。

## 附录：交互状态视觉定义

| 状态 | 视觉变化（基于本规范） |
|------|----------------------|
| Hover | 主按钮背景变为 `--wplus-primary-hover`；次级按钮边框加深或阴影提升 |
| Active / Pressed | 主按钮背景变为 `--wplus-primary-active` |
| Focus | 添加 `2px` 边框，颜色为 `--wplus-primary-border`（品牌金边框） |
| Disabled | 文字透明度降至 `40%` 或使用 `--wplus-text-disabled`，背景变灰，不可交互 |
| Loading | 显示旋转图标，按钮文字可选隐藏或保留，不可再次点击 |
