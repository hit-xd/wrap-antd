# Tabs

**中文名称：** 标签页
**英文名称：** Tabs
**所属分类：** 导航组件
**一句话简介：** 标签（Tabs）是一种横向排列的内容分类/视图切换导航组件，用于将多维度信息按层级拆分，让用户在不离开当前上下文的前提下，快速切换不同内容视图。

---

## 1. 组件定义

标签页用于在同一页面、同一模块或同一数据范围内进行内容切换。当前规范包含四类标签页组件：

1. **一级标签页**：页面内容区域的一级业务域切换，视觉层级最高。
2. **二级标签页**：一级业务域下的二级视图或二级页面切换。
3. **三级标签页**：二级视图下的细分功能、工具或状态筛选。
4. **分段控制器**：最终内容视图、时间口径、展示方式等轻量切换，不承担页面层级导航。

---

## 2. 一级标签页`PrivatePrimaryTabs`

### 2.1 组件定义

一级标签页用于页面内容区的一级业务域切换，视觉层级最高，通常承载页面主内容的一级分类。

### 2.2 使用场景

适用于：

- 页面内容区一级业务域切换。
- 页面主内容的高层级分类。
- 用户进入页面后需要优先识别的主要内容分组。

使用建议：

- 通常数量为 `3–7` 个。
- 用于较稳定的业务分类，不用于频繁变化的筛选项。

禁止场景：

- 不用于时间口径切换，例如“近一周 / 近一个月 / 近一年”。
- 不用于卡片、表格内部的轻量切换。
- 不用于状态筛选。

### 2.3 结构组成

一级标签页由以下部分组成：

- 标签容器。
- 标签项。
- 标签文字。
- 可选前置图标。
- 可选后置说明 icon。
- 底部分割线。
- 当前选中项底部指示线。

### 2.4 状态规则

一级标签页包含以下状态：

- 默认态：未选中、可点击。
- 选中态：当前页面所在的一级分类。
- 禁用态：不可点击。
- hover 态：可点击，鼠标悬停标签页之上

带说明 icon 的标签项：

- 说明 icon 放在标签文字后方。
- 鼠标 hover 到说明 icon 后出现气泡卡片。
- 气泡卡片样式：文字提示 / Tooltip 

### 2.5 尺寸 / 间距 / 颜色 / 交互规则

#### 整体样式

- 整体高度：`48px`
- 宽度：自适应
- 背景：无填充
- 底部分割线高度：`1px`
- 底部分割线颜色：`#000000 15%`
- 标签排列方向：横向
- 标签之间的水平间距：`24px`

#### 单个标签样式

- 标签高度：`48px`
- 标签文字字号：`16px`
- 标签文字字间距：`0px`
- 标签文字行高：`24px`
- 标签宽度：根据文字内容自适应
- 标签左右内边距：`0px`
- 标签上下内边距：`12px`
- 标签文字与前置图标的水平间距：`4px`
- 标签文字与后置说明 icon 的水平间距：`4px`

#### 选中态

- 文字颜色：`#9B7235`
- 文字字重：`Bold` 
- 底部指示线高度：`2px`
- 底部指示线颜色：`#9B7235`
- 前置图标颜色：`#9B7235`
- 后置说明 icon 颜色：`#9B7235`

#### 默认态

- 文字颜色：`#000000 85%`
- 前置图标颜色：`#000000 65%`
- 后置说明 icon 颜色：`#000000 65%`
- 背景：无填充
- 底部指示线：无

#### 禁用态

- 文字颜色：`#000000 25%`
- 图标颜色：`#000000 25%`
- 不显示底部指示线
- 不响应点击

#### 交互规则

- hover 状态文字颜色：`#9B7235`
- hover 状态图标颜色：`#9B7235`
- 点击后切换当前选中项
- 标签较多时是否支持横向滚动：支持

### 2.6 注意事项

- 一级标签页不能被二级标签页、三级标签页或分段控制器替代。
- 基于 Ant Design 调整时，应避免直接使用 antd 默认 `Tabs` 的默认下划线样式，需要覆盖为本规范中的高度、间距、颜色和选中线样式。
- 一级标签页底部分割线需要贯穿标签区域，当前选中线只跟随标签项内容或标签项宽度。
- 如标签含说明 icon，应保证 hover 到 icon 时触发说明气泡，而不是 hover 整个标签都触发。

## 3. 二级标签页`PrivateSecondaryTabs`

### 3.1 组件定义

二级标签页用于一级业务域下的二级视图或二级页面切换，视觉表现为矩形标签按钮组。

### 3.2 使用场景

适用于：

- 一级业务下的二级视图切换。
- 一级页面内部的二级页面切换。
- 比三级标签页更重要、比一级标签页更局部的内容分类。

不适用于：

- 页面主业务域切换。
- 时间口径切换。
- 卡片 / 表格内部的轻量状态筛选。

### 3.3 结构组成

二级标签页由以下部分组成：

- 标签容器。
- 标签项矩形外框。
- 标签文字。
- 可选前置图标。
- 可选后置说明 icon。

### 3.4 状态规则

二级标签页包含以下状态：

- 默认态：未选中、可点击。
- 选中态：当前二级视图。
- 禁用态：不可点击。
- hover 态：可点击，鼠标悬停标签页之上

带说明 icon 的标签项：

- 说明 icon 放在标签文字后方。
- 鼠标 hover 到说明 icon 后出现气泡卡片。
- 气泡卡片样式：文字提示 / Tooltip 

### 3.5 尺寸 / 间距 / 颜色 / 交互规则

#### 整体样式

- 整体高度：`32px`
- 宽度：自适应
- 背景：无填充
- 标签排列方向：横向
- 标签之间的水平间距：`4px`

#### 单个标签样式

- 标签外框形态：矩形
- 标签高度：`32px`
- 标签宽度：根据内容自适应
- 圆角：`2px`
- 标签文字字号：`14px`
- 标签文字字间距：`0px`
- 标签文字行高：`22px`
- 标签左右内边距： `16px`
- 标签上下内边距： `9px`
- 标签文字与前置图标的水平间距：`4px`
- 标签文字与后置说明 icon 的水平间距：`4px`

#### 选中态

- 背景填充色：`#C5A267`
- 边框颜色：无边框
- 文字颜色：`#FFFFFF`
- 文字字重：Bold 
- 前置图标颜色：`#FFFFFF`
- 后置说明 icon 颜色：`#FFFFFF`

#### 默认态

- 背景填充色：无填充
- 边框颜色：`#000000 25%`
- 文字颜色：`#000000 85%`
- 前置图标颜色：`#000000 65%`
- 后置说明 icon 颜色：`#000000 65%`

#### 禁用态

- 背景填充色：无填充
- 边框颜色：`#000000 15%`
- 文字颜色：`#000000 15%`
- 图标颜色：`#000000 15%`
- 不响应点击

#### 交互规则

- hover 状态背景色：无填充
- hover 状态边框颜色：`#9B7235`
- hover 状态文字颜色：`#9B7235`

### 3.6 注意事项

- 二级标签页应独立于一级标签页，不要通过缩放一级标签页得到。
- 基于 Ant Design 调整时，不建议直接使用 antd `Tabs` 的 `type="card"` 默认样式，因为默认圆角、边框、间距和选中色不符合本规范。
- 二级标签页的选中态是“金色填充块”，不能做成一级标签页的下划线样式。
- 如需前置图标或说明 icon，图标尺寸需与 14px 文字视觉居中对齐，图标尺寸为16px16px

## 5. 分段控制器`PrivateSegmentedControl`

### 5.1 组件定义

分段控制器用于最终内容视图、时间口径、展示方式等轻量切换。它不是页面层级导航，不承担一级、二级或三级内容结构切换。

### 5.2 使用场景

适用于：

- 时间口径切换，例如“近一周 / 近一个月 / 近一年”。
- 卡片 / 表格切换。
- 列表 / 宫格切换。
- 同一数据范围内的展示方式切换。
- 已处理 / 待处理等轻量状态切换【需结合具体业务确认】。

典型使用场景：

- 时间口径切换。
- 卡片 / 表格切换。

不适用于：

- 页面一级业务域切换。
- 一级业务下的二级页面切换。
- 复杂多层级内容导航。

### 5.3 结构组成

分段控制器由以下部分组成：

- 外层容器。
- 分段选项。

- 外层容器高度：`34px`

- 选项高度：`28px`
- 选项宽度：根据内容自适应
- 选项圆角：`2px`
- 选项之间的水平间距：`2px`

## 6. Ant Design 二次封装调整建议

### 6.1 组件映射建议

- `PrivatePrimaryTabs`：可基于 Ant Design `Tabs` 二次封装，但需要完全覆盖一级标签页样式。
- `PrivateSecondaryTabs`：可基于 Ant Design `Tabs`、`Radio.Group` 或自定义按钮组封装，但最终视觉必须符合本规范。
- `PrivateTertiaryTabs`：可基于 Ant Design `Tabs`、`Radio.Group` 或自定义按钮组封装，但不得复用二级标签页选中填充样式。
- `PrivateSegmentedControl`：建议基于 Ant Design `Segmented` 二次封装。

### 6.2 不允许的实现结果

- 不允许业务页面直接使用 Ant Design 原生 `Tabs`、`Segmented` 并自行覆盖样式。
- 不允许一级、二级、三级标签页只用同一个组件通过 `type` 参数任意切换，除非组件内部已经封装了明确的业务层级语义。
- 不允许把分段控制器用于页面级导航。
- 不允许把二级标签页做成一级下划线样式。
- 不允许把三级标签页做成二级金色填充样式。
- 不允许在业务页面里写大量 inline style 修正标签页样式。

### 6.3 需要大模型检查的内容

- 检查组件库中是否仍直接导出或推荐使用原生 antd `Tabs` / `Segmented`。
- 检查 demo 页面是否使用业务组件，而不是直接使用 antd 原生组件。
- 检查选中态颜色是否统一使用规范中的金色体系。
- 检查 icon 颜色是否随标签状态变化。
- 检查禁用态是否禁止点击，并使用弱化颜色。
- 检查说明 icon 的 hover 气泡是否只由说明 icon 触发。
- 检查一级、二级、三级、分段控制器是否在视觉上有明确层级差异。
