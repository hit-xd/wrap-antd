# Timeline

**中文名称：** 时间轴
**英文名称：** Timeline
**所属分类：** 数据展示
**一句话简介：** 垂直展示时间流信息。

---

## 组件用法

**核心功能：** 时间轴组件，垂直展示按时间顺序排列的事件。

**基础用法示例：**
```tsx
<Timeline items={[
  { color: 'green', children: '客户回访完成 2026-05-12' },
  { color: 'blue', children: '资产配置方案已提交' },
  { color: 'gray', children: '面谈预约已确认' },
  { color: 'red', children: '风险预警触发' },
]} />
```

**交互说明：**
- `color` 控制时间点颜色（业务语义）
- `pending` 添加待处理节点

---

## 使用场景

**适用场景：**
- 客户经营时间线
- 审批流程、事件记录

**不适用场景：**
- 步骤引导（使用 Steps）

---

## API 文档

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| items | TimelineItem[] | [] | 时间轴项 |
| pending | ReactNode | — | 待处理节点 |
| mode | 'left' \| 'alternate' \| 'right' | — | 排列模式 |

### Events

该组件为纯展示型组件，无事件回调。
