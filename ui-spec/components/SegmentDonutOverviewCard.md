# SegmentDonutOverviewCard

**中文名称：** 图表
**英文名称：** Chart
**所属分类：** 数据展示
**一句话简介：** 环形图概览卡片，展示客户分层或业务占比分布。

---

## 组件用法

**核心功能：** 环形图与概览数据结合的信息卡片，展示客户分层分布（如私行客户、钻石客户、金葵花客户等）及其占比和数值。

**基础用法示例：**
```tsx
<SegmentDonutOverviewCard
  title="客户分层分布"
  segments={[
    { label: '私行客户', value: 1286, color: '#C5A267' },
    { label: '钻石客户', value: 942, color: '#1874FF' },
    { label: '金葵花客户', value: 568, color: '#00C078' },
  ]}
/>
```

**交互说明：**
- 环形图展示各分层占比
- 数据项展示具体数值


## 字体说明

- 正文说明、标签、操作按钮和常规辅助信息默认使用 14px。
- 核心数字、统计值和标题按组件语义单独定义字号，不与正文混用。

---

## 使用场景

**适用场景：**
- 客户分层分布展示
- 业务占比概览卡片

**不适用场景：**
- 纯图表展示（使用独立的 Chart 组件）

---

## API 文档

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| title | ReactNode | — | 卡片标题 |
| segments | Segment[] | [] | 数据分段配置 |

### Events

该组件为纯展示型组件，无事件回调。
