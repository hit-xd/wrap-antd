# DashboardPageTemplate

**中文名称：** 经营看板模板
**英文名称：** DashboardPageTemplate
**所属分类：** 典型页面
**一句话简介：** 经营看板页面模板，展示中心负责人的关键经营指标与数据看板。

---

## 组件用法

**核心功能：** 经营看板页面模板，包含顶部指标条（SummaryStatStrip）、图表卡片（PrivateChartCard）、排行榜表格和待办任务区等典型看板模块。

**交互说明：**
- 顶部汇总指标条展示管户数、AUM、新增客户、待处理任务
- 中部图表区展示 AUM 趋势、产品配置分布
- 右侧经理排行榜展示团队业绩


## 字体说明

- 页面中的筛选、工具栏、表格、分页、标签和辅助文案默认使用 14px。
- 页面级标题、统计数字和反馈标题按语义单独定义字号。

---

## 使用场景

**适用场景：**
- 中心负责人、团队管理者的经营看板
- 需要汇总指标+图表+排名的数据 dashboard
