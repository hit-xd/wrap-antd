# Mentions

**中文名称：** 提及
**英文名称：** Mentions
**所属分类：** 数据录入
**一句话简介：** 提及组件，在输入中 @ 引用人员或内容。

---

## 组件用法

**核心功能：** 提及组件，在文本输入中通过 @ 符号快速引用人员、机构或其他实体。

**基础用法示例：**
```tsx
<Mentions
  placeholder="输入 @ 提及人员"
  options={[
    { value: '张三', label: '张三' },
    { value: '李四', label: '李四' },
  ]}
/>
```

**交互说明：**
- 输入 @ 自动弹出候选列表
- 支持自定义触发字符


## 字体说明

- 字段标签、输入值、下拉选项、日期值、单选/多选项与操作按钮默认使用 14px。
- 占位提示、校验反馈或补充说明如有单独字号要求，按 Typography 规范执行。

---

## 使用场景

**适用场景：**
- 评论、协同输入中提及人员
- 工作流备注中引用相关方

**不适用场景：**
- 普通文本输入（使用 Input）

## API 文档

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| value | string | — | 输入框值 |
| options | { value, label }[] | [] | 提及候选列表 |
| placeholder | string | — | 占位符 |
| prefix | string \| string[] | '@' | 触发字符 |

### Events

| 事件名 | 回调参数 | 触发时机 |
|--------|----------|----------|
| onChange | (text) => void | 内容变化时触发 |
| onSelect | (option) => void | 选中选项时触发 |
