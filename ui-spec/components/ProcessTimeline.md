# ProcessTimeline

**中文名称：** 审批流程
**英文名称：** ProcessTimeline
**所属分类：** 导航组件
**一句话简介：** 垂直审批流程时间线，以时间线形式展示业务审批或任务进度，支持五种节点状态。

---

## 组件用法

### 审批流程

**核心功能：** 以垂直排列的节点和贯穿式竖线，按审批时间顺序展示流程节点、流转状态和审批信息，帮助用户清晰感知审批进度、历史操作和当前状态。

**五种节点状态：**
- **已完成**（`finish`）：绿色圆形 + 白色对勾，绿色竖线
- **待处理**（`process`）：蓝色圆形 + 三点标识，灰色竖线，右侧蓝色"待处理"标签
- **未激活**（`wait`）：浅灰色空心圆形，灰色竖线
- **已回退**（`return`）：红色圆形 + 回退箭头，红色竖线，右侧红色"已回退"标签
- **已驳回**（`reject`）：红色圆形 + 白色叉号，红色竖线（流程终点），右侧红色"已驳回"标签

**基础用法示例：**
```tsx
import { ApprovalTimeline } from 'wplus-design';

export default function Example() {
  return (
    <ApprovalTimeline
      title="开户审批流程"
      items={[
        {
          title: '王源/032345',
          department: '总行/办公室/综合调研室',
          time: '09-03 10:18',
          status: 'finish',
          extra: '同意申请审批意见。',
        },
        {
          title: '李芳/045678',
          department: '风险管理部',
          time: '09-03 14:22',
          status: 'finish',
          extra: '客户资料已核实，风险等级评估完成。',
        },
        {
          title: '张振/056789',
          department: '风控中心',
          time: '09-04 09:15',
          status: 'process',
          tag: '待处理',
        },
        {
          title: '陈敏/067890',
          department: '合规管理部',
          status: 'wait',
        },
        {
          title: '赵立/078901',
          department: '总行办公室',
          status: 'wait',
        },
      ]}
    />
  );
}
```

**交互说明：**
- 竖线颜色与节点状态主色一致，已完成/已回退/已驳回节点竖线为对应状态色，待处理/未激活节点竖线为中性灰色
- 已驳回节点为流程终点，竖线终止于该节点，后续无内容
- 节点右侧信息区包含审批人/工号（14px，500字重）、所属部门（13px，次要文本色）、操作时间（13px，次要文本色，居右对齐）
- 已完成/已回退/已驳回节点的附加信息以浅灰色背景容器展示，待处理/未激活节点无附加信息

---

## 字体说明

- 审批人/工号：14px，主文本色，字重 500
- 所属部门：13px，次要文本色
- 操作时间：13px，次要文本色
- 状态标签：12px，500字重，颜色与状态色一致
- 附加信息：13px，次要文本色

---

## 使用场景

**适用场景：**
- 开户审批、资料审核、业务申请等流程进度追踪
- 审批流程进度、任务处理进度追踪、多步骤历史记录
- 展示审批被回退、驳回等异常流程状态

**不适用场景：**
- 水平步骤导航（使用 NavigationSteps）
- 纯时间线（使用 Timeline）

## API 文档

### ApprovalTimeline Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| title | ReactNode | — | 审批流程标题 |
| items | ApprovalTimelineItem[] | 必填 | 审批节点数组 |

### ApprovalTimelineItem

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| title | string | — | 审批人姓名/工号 |
| department | string | — | 所属部门 |
| time | string | — | 操作时间 |
| status | 'finish' \| 'process' \| 'wait' \| 'return' \| 'reject' | — | 节点状态：finish=已完成、process=待处理、wait=未激活、return=已回退、reject=已驳回 |
| tag | string | — | 状态标签文字（如"待处理""已回退""已驳回"） |
| extra | ReactNode | — | 附加信息（审批意见/驳回原因），仅已完成/已回退/已驳回节点展示 |
