# Menu

**中文名称：** 导航菜单
**英文名称：** Menu
**所属分类：** 导航组件
**一句话简介：** 为页面和功能提供导航菜单。涵盖顶部导航、侧边导航及业务页导航组合等多种形态。

---

# 一、顶部导航菜单

## 1. 组件定义

导航菜单用于承载系统级导航能力，帮助用户在不同业务页面、业务模块和功能层级之间切换。

顶部导航菜单对应 Ant Design `Menu` 的横向导航形态，可基于 `mode="horizontal"` 二次封装。

顶部导航菜单主要用于私行 W+ 系统顶部一级业务模块切换。

---

## 2. 使用场景

顶部导航菜单适用于系统全局一级导航，用于承载用户在不同业务域之间的跳转。

适用场景包括：

- 首页
- 客户管理
- 商机管理
- 工作管理
- 产品管理
- 资产配置
- 双录服务
- 全球连线
- 陪伴一站通
- 分行专区
- 更多

顶部导航通常固定在页面顶部，用于表达系统级主导航，不建议承载过深层级或过多局部筛选项。

---

## 3. 结构组成

顶部导航菜单由以下区域组成：

1. 左侧私行 W+ Logo 区
2. 中部导航标签区
3. 右侧个人信息区

### 3.1 左侧 Logo 区

Logo 区用于展示私行 W+ 品牌标识。

结构包括：

- 私行 W+ logo
- logo 外侧留白

样式要求：

- Logo 尺寸：`100px × 32px`
- Logo 距离容器左侧：`24px`
- Logo 上下边距：`16px`
- Logo 与第一个导航标签之间的固定间距：`88px`

### 3.2 中部导航标签区

中部导航标签区用于展示系统一级导航标签。

结构包括：

- 导航文字
- 选中态底部描边
- 导航项之间的水平间距

样式要求：

- 导航标签字体字号：`16px`
- 导航标签行高：`24px`
- 导航标签字间距：`0px`
- 导航标签上下边距：`20px`
- 导航标签左右边距：`0px`
- 导航标签之间的水平间距：`24px`
- 最后一个导航标签与右侧个人信息模块之间的最小间距：`88px`

### 3.3 右侧个人信息区

右侧个人信息区用于展示当前登录用户信息。

结构包括：

- 用户头像
- 姓名 / 工号
- 角色名称
- 三角形展开 icon

样式要求：

- 个人信息区整体高度：`44px`
- 个人信息区宽度：自适应
- 个人信息区距离顶部导航容器右侧：`24px`
- 头像尺寸：`28px × 28px`
- 头像与姓名/角色信息组之间的水平间距：`12px`
- 姓名 / 工号与三角形 icon 之间的水平间距：`6px`
- 三角形 icon 水平居中对齐

姓名 / 工号文字：

- 字号：`14px`
- 行高：`22px`
- 字重： `Regular `
- 颜色： `#FFFFFF `

角色名称文字：

- 字号：`12px`
- 行高：`20px`
- 字重：`Regular `
- 字间距：`0px`
- 颜色：`#FFFFFF 65% `

姓名 / 工号单行展示，角色名称单行展示，两者上下间距：`2px`，整体左对齐。

---

## 4. 状态规则

### 4.1 默认状态

默认状态用于未选中的顶部导航标签。

样式要求：

- 文字颜色：#FFFFFF 65%
- 背景色：透明 
- 底部描边：无
- 字重：Regular

### 4.2 选中状态

选中状态用于当前所在的顶部一级导航。

样式要求：

- 文字颜色： `#FFEBAE`
- 字重：加粗
- 底部描边高度：`2px`
- 底部描边颜色： `#FFEBAE`
- 底部描边与顶部导航容器底部对齐

### 4.3 Hover 状态

Hover 状态用于鼠标悬停在顶部导航标签上时。

样式要求：

- 文字颜色： `#FFEBAE`
- 背景色：透明
- 是否显示底部描边：不显示
- 字重：Regular

### 4.4 禁用状态

禁用状态用于不可点击的顶部导航标签。

样式要求：

- 文字颜色：#FFFFFF 65%
- 是否响应 hover：不响应

---

## 5. 尺寸 / 间距 / 颜色 / 交互规则

### 5.1 顶部导航尺寸规则

顶部导航整体：

- 高度：`64px`
- 宽度：自适应，源图示例宽度为 `1980px`
- 背景色：`#0C0C18`
- 下边框：无
- 外层描边：无

左侧 Logo：

- Logo 尺寸：`100px × 32px`
- 左边距：`24px`
- 上边距：`16px`
- 下边距：`16px`

中部导航：

- 字号：`16px`
- 行高：`24px`
- 上下边距：`20px`
- 左右边距：`0px`
- 导航项之间水平间距：`24px`
- 第一个导航项距离 Logo：`88px`
- 最后一个导航项距离右侧个人信息模块最小距离：`88px`

右侧个人信息：

- 模块高度：`44px`
- 宽度：自适应
- 右边距：`24px`
- 头像尺寸：`28px × 28px`
- 头像与文字组间距：`12px`
- 姓名 / 工号与角色名称垂直间距：`2px`
- 文字组与三角 icon 间距：`6px`

### 5.2 顶部导航颜色规则

顶部导航容器背景色： `#0C0C18`

默认导航文字：#FFFFFF 65%

选中导航文字：`#FFEBAE`

选中导航底部描边：

- 高度：`2px`
- 颜色：`#FFEBAE`

Logo ：

- 直接引用icon图标库中的私行W+logo图标

### 5.3 顶部导航交互规则

- 点击导航标签后切换当前页面或业务模块。
- 当前所在导航标签必须展示选中态。
- 导航标签不应出现 Ant Design 默认蓝色 hover / active 样式。
- 导航标签之间的间距应固定为 `24px`，不要使用 Ant Design 默认 padding 造成不可控宽度。
- 个人信息区域点击后是否展开个人菜单：不展开
- “更多”导航点击后是否展示下拉菜单：不展开

### 5.4 顶部导航基于 Ant Design Menu 的调整建议

大模型 调整时建议重点覆盖以下 Ant Design 默认样式：

- 覆盖 `.ant-menu-horizontal` 背景色为 `#0C0C18`
- 移除 `.ant-menu-horizontal` 默认底部边框
- 移除 Ant Design 默认蓝色选中线
- 移除 Ant Design 默认蓝色 hover / active 色
- 将导航项高度固定为 `64px`
- 将导航文字行高固定为 `24px`
- 将导航项上下边距按 `20px` 处理
- 将导航项间距固定为 `24px`
- 选中态底部描边高度固定为 `2px`
- 选中态底部描边贴齐顶部导航容器底部
- Logo 区和个人信息区不要作为普通 `Menu.Item` 参与菜单选中逻辑

---

## 6. 注意事项

1. 顶部导航和侧边导航虽然都属于导航菜单，但不要共用同一套颜色和选中态逻辑。
   - 顶部导航是深色背景，选中态适合使用浅金色。

2. 不建议直接使用 Ant Design 默认 Menu 样式。
   - 默认蓝色选中态、hover 态和缩进规则都需要覆盖。
   - 应使用私行组件库中的业务组件承载导航样式。

3. 顶部 Logo 区和右侧个人信息区不应作为普通菜单项处理。
   - 它们属于导航容器内的固定区域。
   - 中间导航标签才参与 Menu 的 active / selected 逻辑。

4. 选中态不要新增额外高度。
   - 选中态只改变文字颜色、字重、背景色和右侧描边。
   - 不应导致导航项高度、宽度或布局跳动。

5. 当前 SVG 中部分文字标注与视觉采样存在轻微差异。
   - 顶部导航选中态源图视觉更接近 `#FFEBAE`。
   - 源图文字说明中出现 `#9B7235`。
   - 该色值建议由设计师最终确认后再写入组件库 token。

6. 若后续组件库已有 Design Token，应优先将以下颜色抽为 token：
   - 顶部导航背景：`#0C0C18`
   - 顶部导航默认文字：`rgba(255,255,255,0.65)`
   - 顶部导航选中色：【待确认，建议 `#FFEBAE`】

# 二、侧边导航菜单

## 1. 组件定义

导航菜单用于承载系统级导航能力，帮助用户在不同业务页面、业务模块和功能层级之间切换。

侧边导航菜单对应 Ant Design `Menu` 的侧边导航形态，可基于 `mode="inline"` 二次封装。

侧边导航菜单主要用于高密度业务系统中的多级功能导航，当前规范支持一级、二级、三级导航。

---

## 2. 使用场景

侧边导航菜单适用于高密度操作界面和沉浸式作业系统，适合在页面左侧承载多级功能导航，最大化利用横向空间。

适用场景包括：

- 工作台类页面
- 管理后台类页面
- 多级业务功能页面
- 需要长期保持左侧导航可见的业务系统

侧边导航当前支持三级结构：

- 一级导航
- 二级导航
- 三级导航

---

## 3. 结构组成

侧边导航菜单由以下层级组成：

1. 一级导航
2. 二级导航
3. 三级导航

### 3.1 侧边导航整体结构

侧边导航整体样式：

- 侧边导航宽度：`208px`
- 侧边导航高度：自适应
- 背景色：`#FFFFFF`
- 描边：无
- 一共支持三级导航
- 每个导航标签容器宽度：`208px`
- 每个导航标签容器高度：`40px`

### 3.2 一级导航结构

一级导航用于表达最高层级的功能分组。

结构包括：

- 前置图标
- 一级导航文字
- 展开 / 收起 icon

样式要求：

- 容器宽度：`208px`
- 容器高度：`40px`
- 内容在容器中水平居中对齐
- 前置图标尺寸：`16px × 16px`
- 一级导航文字字号：`14px`
- 一级导航文字容器宽度：`124px`
- 前置图标距离容器左侧：`16px`
- 前置图标与文字之间的水平间距：`12px`
- 文字与展开 / 收起 icon 之间的水平间距：`8px`

### 3.3 二级导航结构

二级导航用于承载一级导航下的功能分组。

结构包括：

- 二级导航文字
- 可选展开 / 收起 icon
- 选中态右侧描边

样式要求：

- 容器宽度：`208px`
- 容器高度：`40px`
- 二级导航前面不带 icon
- 当二级导航下方有三级导航时，后面带展开 / 收起 icon
- 二级导航文字容器宽度：`124px`
- 二级导航距离容器左侧：`44px`
- 二级导航文字字号：`14px`
- 选中态最右侧描边宽度：`2px`
- 选中态最右侧描边高度：`40px`
- 选中态最右侧描边颜色：`#9B7235`
- 选中态最右侧描边与容器右侧对齐

### 3.4 三级导航结构

三级导航用于承载最末级功能入口。

结构包括：

- 三级导航文字
- 选中态右侧描边

样式要求：

- 容器宽度：`208px`
- 容器高度：`40px`
- 三级导航为纯文字
- 三级导航前面不带 icon
- 三级导航后面不带展开 / 收起 icon
- 三级导航文字容器宽度：`136px`
- 三级导航距离容器左侧：`56px`
- 三级导航右侧边距：`16px`
- 三级导航文字字号：`14px`
- 选中态最右侧描边宽度：`2px`
- 选中态最右侧描边高度：`40px`
- 选中态最右侧描边颜色：`#9B7235`
- 选中态最右侧描边与容器右侧对齐

---

## 4. 状态规则

### 4.1 未选中状态

未选中状态用于非当前导航项。

样式要求：

- 文字颜色：#000000 65%
- 如有图标，图标颜色：#000000 65%
- 背景色：`#FFFFFF`
- 字重：Regular

### 4.2 Hover 状态

Hover 状态用于鼠标悬停在侧边导航项上时。

样式要求：

- 文字颜色：`#9B7235`
- 字重：Regular
- 如有图标，图标颜色：`#9B7235`
- 背景色：`#FFFFFF`

### 4.3 选中状态

选中状态用于当前所在的侧边导航项。

样式要求：

- 文字颜色：`#9B7235`
- 字重：Bold
- 如有图标，图标颜色：`#9B7235`
- 背景色：`#FFFCF0`
- 右侧选中描边宽度：`2px`
- 右侧选中描边高度：`40px`
- 右侧选中描边颜色：`#9B7235`
- 右侧选中描边与容器右侧对齐

### 4.4 展开状态

展开状态用于有子级导航的一级或二级导航。

样式要求：

- 展开 / 收起 icon 位于导航项右侧
- 展开 / 收起 icon 与前方文字保持 `8px` 间距
- 展开状态 icon 方向：向上
- 展开状态 icon 颜色：`#9B7235`
- 收起状态 icon 方向：向下
- 收起状态 icon 颜色：#000000 65% 

### 4.5 禁用状态

禁用状态用于不可点击的侧边导航项。

样式要求：

- 文字颜色：#000000 15% 
- 图标颜色：#000000 15% 
- 背景色：`#FFFFFF`
- 是否响应 hover：不响应

---

## 5. 尺寸 / 间距 / 颜色 / 交互规则

### 5.1 侧边导航尺寸规则

侧边导航整体：

- 宽度：`208px`
- 高度：自适应
- 背景色：`#FFFFFF`
- 每个导航项容器宽度：`208px`
- 每个导航项容器高度：`40px`

一级导航：

- 前置图标尺寸：`16px × 16px`
- 一级文字字号：`14px`
- 一级文字容器宽度：`124px`
- 左侧图标距离容器左侧：`16px`
- 前置图标与文字间距：`12px`
- 文字与展开 / 收起 icon 间距：`8px`

二级导航：

- 文字字号：`14px`
- 文字容器宽度：`124px`
- 距离容器左侧：`44px`
- 选中态右侧描边宽度：`2px`
- 选中态右侧描边高度：`40px`

三级导航：

- 文字字号：`14px`
- 文字容器宽度：`136px`
- 距离容器左侧：`56px`
- 右侧边距：`16px`
- 选中态右侧描边宽度：`2px`
- 选中态右侧描边高度：`40px`

### 5.2 侧边导航颜色规则

未选中状态：

- 文字颜色：#000000 65%
- 图标颜色：#000000 65%
- 背景色：`#FFFFFF`

Hover 状态：

- 文字颜色：`#9B7235`
- 图标颜色：`#9B7235`
- 背景色：`#FFFFFF`

选中状态：

- 文字颜色：`#9B7235`
- 图标颜色：`#9B7235`
- 背景色：`#FFFCF0`
- 右侧描边颜色：`#9B7235`

### 5.3 侧边导航交互规则

- 点击一级导航时，如存在子级，则展开或收起下级导航。
- 点击二级导航时，如存在三级导航，则展开或收起三级导航。
- 点击三级导航时，切换页面并展示选中状态。
- 仅当前页面的导航展示选中态背景色`#FFFCF0` 右侧描边为2px、颜色`#9B7235`；关联的母级导航的背景色为#FFFFFF。
- 有子级的导航项应显示展开 / 收起 icon。
- 无子级的三级导航不显示展开 / 收起 icon。
- 选中态应优先作用在当前实际页面所在的最低层级导航项。
- 当父级包含当前选中子项时，父级是否也需要展示选中态：需要

### 5.4 侧边导航基于 Ant Design Menu 的调整建议

大模型 调整时建议重点覆盖以下 Ant Design 默认样式：

- 覆盖 `.ant-menu-inline` 宽度为 `208px`
- 移除 Ant Design 默认蓝色选中背景和蓝色选中竖线
- 移除 Ant Design 默认 `inline-indent` 带来的不可控缩进
- 一级、二级、三级缩进使用本规范中的固定距离
- 所有导航项高度统一为 `40px`
- 一级、二级、三级文字字号统一为 `14px`
- 选中态背景色统一为 `#FFFCF0`
- 选中态文字和图标统一为 `#9B7235`
- 选中态右侧描边统一为 `2px × 40px`
- 展开 / 收起 icon 位置需与文字右侧保持 `8px` 间距
- 避免使用 Ant Design 默认 `padding-left` 自动计算层级缩进

---

## 6. 注意事项

1. 顶部导航和侧边导航虽然都属于导航菜单，但不要共用同一套颜色和选中态逻辑。
   - 侧边导航是白色背景，选中态使用 `#9B7235` 和 `#FFFCF0`。

2. 不建议直接使用 Ant Design 默认 Menu 样式。
   - 默认蓝色选中态、hover 态和缩进规则都需要覆盖。
   - 应使用私行组件库中的业务组件承载导航样式。

3. 侧边导航层级必须保持清晰。
   - 一级导航可带前置图标和展开 / 收起 icon。
   - 二级导航不带前置图标，可在存在三级菜单时带展开 / 收起 icon。
   - 三级导航为纯文字，不带图标和展开 / 收起 icon。

4. 选中态不要新增额外高度。
   - 选中态只改变文字颜色、字重、背景色和右侧描边。
   - 不应导致导航项高度、宽度或布局跳动。
