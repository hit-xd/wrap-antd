# Anchor

**中文名称：** 锚点
**英文名称：** Anchor
**所属分类：** 导航组件
**一句话简介：** 锚点组件用于页面内长内容区域的快速定位导航。用户点击锚点项后，页面滚动至对应内容模块；当页面滚动到对应模块时，锚点项需要同步高亮当前所在位置。

---

## 何时使用

适用于以下场景：

- 详情页中多个信息区块的快速定位。
- 表单页中多个分组模块的跳转。
- 长页面中的局部导航，例如“基础信息 / 业务信息 / 附件信息 / 操作记录”。
- 文档型页面、设置页、配置页中需要随页面滚动同步定位的场景。

不建议用于以下场景：

- 页面级主导航。
- 多页面之间的跳转导航。
- 替代 Tabs 的内容切换。
- 锚点项数量过多且无滚动容器的场景。

## 3. 结构组成

组件由以下部分组成：

1. 外层容器
2. 锚点项列表
3. 左侧竖向引导线
4. 当前选中圆点指示器
5. 锚点文字

当前设计图示例包含 4 个锚点项：

- 导航
- 数据录入
- 数据展示
- 反馈

### 3.1 外层容器

外层容器宽度：`200px`  
外层容器高度：根据锚点项数量自适应，当前 4 项示例高度为 `148px`  
外层容器背景色：`#FFFFFF`  
外层容器描边：无  
外层容器圆角：无   
外层容器阴影：无

### 3.2 锚点项

锚点项排列方向：竖向  
单个锚点项高度：`22px`  
锚点项之间的垂直间距：`12px`  
首个锚点项距离容器顶部：`12px`  
最后一个锚点项距离容器底部：`12px`  
锚点项宽度：跟随容器宽度，当前示例为 `200px`

### 3.3 左侧引导线

左侧引导线宽度：`2px`  
左侧引导线颜色：`#F4F6F8`  
左侧引导线距离容器左侧：`13px`  
左侧引导线与当前选中圆点中心对齐  
左侧引导线不随选中状态变色  
左侧引导线圆角：无

### 3.4 当前选中圆点指示器

选中圆点外层尺寸：`8px × 8px`  
选中圆点形态：圆形描边点  
选中圆点外圈颜色：`#9B7235`  
选中圆点中心颜色：`#FFFFFF`  
选中圆点外圈厚度：约 `2px`  
选中圆点圆角：`4px`  
选中圆点中心与左侧引导线中心对齐

### 3.5 锚点文字

锚点文字位于左侧引导线右侧  
锚点文字左侧起始位置：距离容器左侧约 `30px`  
选中圆点右边缘与文字之间的水平间距：约 `12px`  
锚点文字垂直方向与锚点项居中对齐  
锚点文字左对齐

## 4. 状态规则

锚点组件至少包含以下状态：

1. 默认状态
2. 当前选中状态
3. 悬停状态
4. 聚焦状态
5. 禁用状态


### 4.1 默认状态

默认状态用于非当前所在位置的锚点项。

默认状态不显示圆点指示器  
默认状态左侧仅显示浅色竖向引导线  
默认状态文字颜色：`#000000 85%`  
默认状态文字字号：14px  
默认状态文字字重：Regular  
默认状态文字行高：22px  

### 4.2 当前选中状态

当前选中状态用于表示页面当前滚动所在区域，或用户点击后定位到的锚点项。

当前选中状态显示圆形指示器  
当前选中圆点尺寸：`8px × 8px`  
当前选中圆点外圈颜色：`#9B7235`  
当前选中圆点中心颜色：`#FFFFFF`  
当前选中圆点与左侧引导线中心对齐

当前选中文字颜色：`#9B7235`  
默认状态文字字号：14px  
默认状态文字字重：Regular  
默认状态文字行高：22px  

### 4.3 悬停状态

悬停状态文字颜色：`#9B7235`  
悬停状态是否显示圆点：不显示
悬停状态左侧引导线颜色：`#F4F6F8`  
悬停状态鼠标样式：`pointer`

## 5. 尺寸 / 间距 / 颜色 / 交互规则

### 5.1 尺寸规则

外层容器宽度：`200px`  
外层容器高度：根据锚点项数量自适应，当前 4 项为 `148px`  
单个锚点项高度：`22px`  
左侧引导线宽度：`2px`  
当前选中圆点尺寸：`8px × 8px`

### 5.2 间距规则

锚点项之间垂直间距：`12px`  
首个锚点项距离容器顶部：`12px`  
最后一个锚点项距离容器底部：`12px`  
左侧引导线距离容器左侧：`13px`  
锚点文字距离容器左侧：约 `30px`  
当前选中圆点右边缘与文字之间的水平间距：约 `12px`

### 5.3 颜色规则

容器背景色：`#FFFFFF`  
左侧引导线颜色：`#F4F6F8`  
当前选中主色：`#9B7235`  
当前选中文字颜色：`#9B7235`  
当前选中圆点外圈颜色：`#9B7235`  
当前选中圆点中心颜色：`#FFFFFF`  
默认文字颜色：`rgba(0, 0, 0, 0.85)` / `#000000 85%`

### 5.4 文字规则

锚点文字字号：`14px`
锚点文字字重：Regular 
锚点文字行高：`22px`
锚点文字对齐方式：左对齐  
锚点文字垂直对齐方式：与锚点项居中对齐

### 5.5 交互规则

点击锚点项后，页面滚动至对应内容区块。  
页面滚动到对应内容区块时，锚点项需要同步进入当前选中状态。  
同一时间只允许一个锚点项处于当前选中状态。  
默认状态锚点项可点击。  
当前选中项仍可点击，但点击后不改变视觉状态。  
滚动同步时，选中圆点应跟随当前锚点项切换。

## 6. 基于 Ant Design Anchor 的样式调整建议

组件库基于 Ant Design 构建时，可基于 `Anchor` 二次封装，但需要覆盖 Ant Design 默认样式。

### 6.1 需要覆盖的 Ant Design 默认结构

需要重点调整以下结构或等价类名：

- `.ant-anchor`
- `.ant-anchor-wrapper`
- `.ant-anchor-link`
- `.ant-anchor-link-title`
- `.ant-anchor-ink`
- `.ant-anchor-ink-ball`

### 6.2 覆盖要求

Ant Design 默认的锚点线条、选中样式、文字颜色需要替换为本规范样式。  
不要直接使用 Ant Design 默认蓝色选中态。  
不要保留 Ant Design 默认 active 状态的蓝色线条。  
不要让锚点文字在 active 状态出现下划线。  
不要让左侧引导线随 active 状态变色。  
不要保留Ant Design 选中态的蓝色描边，当前选中态要用空心圆点，颜色为 `#9B7235`，并与左侧浅色引导线中心对齐。

## 7. 注意事项

1. 锚点组件是页面内定位导航，不是页面级菜单，也不是 Tabs。
2. 锚点只用于长页面内容定位，不用于控制内容显隐切换。
3. 当前选中状态必须与页面滚动位置同步，不应只依赖点击状态。
4. 左侧竖线是弱引导元素，颜色固定为 `#F4F6F8`，不要加深为主色。
5. 选中状态只通过圆点和文字主色表达，不额外增加背景块。
6. 默认状态不显示圆点，避免视觉干扰。
7. 如果锚点项较多，建议在外层容器中增加滚动能力，滚动样式需另行补充。
8. 如果用于右侧固定锚点导航，需要补充固定定位、距离页面右侧、顶部偏移等布局规则。
9. 如果页面顶部存在固定 Header，需要配置滚动偏移量，避免定位后内容被 Header 遮挡。
10. 该组件样式应在组件库内部封装，业务页面不应单独覆盖 Ant Design Anchor 样式
