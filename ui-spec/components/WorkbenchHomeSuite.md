# WorkbenchHomeSuite

**中文名称：** 首页
**英文名称：** WorkbenchHomeSuite
**所属分类：** 典型页面
**一句话简介：** 私行首页案例，完整展示顶部导航、居中展示区、首页搜索框、图表卡片、快捷入口与待办区。

---

## 组件用法

**核心功能：** 私行工作台首页的完整示例，组合 WorkbenchHomeHeader、HomeSearch、MetricCard、QuickEntryGrid、GroupedTodoBoard 等组件，展示首页工作台的标准布局。

**交互说明：**
- 首页居中布局，顶部为品牌导航栏
- 搜索框支持全局搜索
- 指标卡展示关键业务数据
- 快捷入口提供高频操作导航


## 字体说明

- 页面中的筛选、工具栏、表格、分页、标签和辅助文案默认使用 14px。
- 页面级标题、统计数字和反馈标题按语义单独定义字号。

---

## 使用场景

**适用场景：**
- 私行系统首页参考
- 工作台类页面的标准布局

---

## API 文档

### Props

该组件为页面级组合示例，无独立 Props 接口。

### Events

该组件为纯展示型组件，无事件回调。
