# ImageRatio 图片比例

**中文名称：** 图片比例
**英文名称：** ImageRatio
**所属分类：** 全局样式
**一句话简介：** 统一招行零售 Web 端图片比例标准，定义 1:1、4:3、3:2、16:9、2:1 五类核心比例及其适用场景。

---

## 何时使用

当页面中需要约束图片、视频封面、业务卡片封面、产品封面、客户头像、图文卡片、营销素材、内容缩略图、系统占位图、运营 banner 等视觉素材的固定裁切比例时使用。统一的图片比例规范确保页面内图片裁切和信息密度稳定，避免因素材比例不一致导致的布局错乱和视觉失衡。

建议：同一页面内同类图片应使用统一比例。图片比例应根据内容场景选择，不建议随意拉伸或压缩图片。图片展示区域应预先固定宽高比例，避免图片加载过程中页面布局抖动。当图片内容与容器比例不一致时，应优先保证主体内容完整，其次再考虑视觉铺满。


该规范定义 1:1、4:3、3:2、16:9、2:1 五类核心比例及其适用场景。

---

## 1:1 正方形比例

1:1 正方形比例用于对称、稳定、聚焦的视觉场景，适合需要用户视线集中在小范围区域的内容展示。

该比例为宽高比 1:1，即宽与高相等。容器宽度自适应，高度由比例约束自动计算。当用于头像时，图片可结合 50% 圆角形成圆形头像。当用于卡片缩略图时，图片主体应居中展示，避免重要内容贴边或被裁切。

默认实现通过 CSS `aspect-ratio: 1 / 1` 控制，兼容旧浏览器时使用 `padding-bottom: 100%`。当使用 `padding-bottom` 方式时，容器内内容需通过 `position: absolute` 定位。

适用于用户头像、客户头像、产品图标、业务入口图、宫格卡片图片、小型封面缩略图的场景。

建议：如果图片中包含人物、IP 形象或核心图标，应保证主体位于画面中心区域。当图片来源不统一时，应使用统一容器承载，保证列表或宫格排版整齐。

---

## 4:3 经典矩形比例

4:3 经典矩形比例用于均衡、舒适的视觉场景，是内容展示中最通用的比例，适合大多数常规图片和卡片封面。

该比例为宽高比 4:3，宽度略大于高度，呈现较稳重舒适的矩形视觉感受。图片主体应位于画面中部，避免上下或左右裁切影响识别。同一组卡片如使用 4:3 图片，应保持图片区域高度一致。

默认实现通过 CSS `aspect-ratio: 4 / 3` 控制，兼容旧浏览器时使用 `padding-bottom: 75%`。

适用于内容卡片封面、业务说明插图、产品介绍图片、运营活动图片、列表卡片封面、证件材料预览等需要展现主体内容但不过分拉伸的图片场景。

建议：当页面空间较充足，且图片需要表达一定内容细节时，可优先使用 4:3。如果图片更偏横向信息展示，可考虑使用 3:2 或 16:9。

---

## 3:2 黄金矩形比例

3:2 黄金矩形比例用于修长、协调的视觉场景，比 4:3 更修长，适合故事性较强的图片展示，兼顾横向展示和画面稳定性。

该比例为宽高比 3:2，宽度与高度的比例介于 4:3 和 16:9 之间，呈现修长协调的视觉感受。3:2 图片应避免过度裁切上下内容。当图片中包含人物、产品或关键元素时，应预留安全边距。同一页面内，3:2 图片不应与 4:3 图片混排在同一级卡片中。

默认实现通过 CSS `aspect-ratio: 3 / 2` 控制，兼容旧浏览器时使用 `padding-bottom: 66.6667%`。

适用于客户故事、活动图片、产品封面、业务专题图、营销内容配图、中型 Banner 模块、图文列表缩略图等需要视觉修长感和叙事性的场景。

建议：3:2 适合用于卡片宽度较大、但不希望图片过高的场景。如果内容更偏展示氛围或横向视觉，可使用 16:9 替代。

---

## 16:9 宽屏矩形比例

16:9 宽屏矩形比例用于宽屏、冲击力的视觉场景，是目前最广泛的宽屏标准比例，适合营造沉浸式视觉体验。

该比例为宽高比 16:9，宽度显著大于高度，呈现宽广的横向视野。图片中的主体元素不宜靠近左右边缘，避免在不同屏幕下被裁切。当作为 Banner 使用时，应提前定义安全区域，确保文字、Logo 和核心视觉不被遮挡。

默认实现通过 CSS `aspect-ratio: 16 / 9` 控制，兼容旧浏览器时使用 `padding-bottom: 56.25%`。

适用于视频封面、页面 Banner、客户建议书展示图、系统首页头图、运营活动横幅、数据看板配图、工作台重点图等需要突出视觉冲击力和宽屏沉浸感的场景。

建议：16:9 适合强调氛围和视觉冲击，但不适合承载过多细节文字。如图片上叠加文字，应增加遮罩或渐变层，保证文字可读性。移动端使用 16:9 时，应关注图片高度是否过低，避免主体内容不清晰。

---

## 2:1 超宽矩形比例

2:1 超宽矩形比例用于横向、广视野的视觉场景，是五类比例中最宽的比例，适合扩大信息承载量。不适合展示复杂内容。核心元素应尽量居中或靠近视觉焦点区域。如图片中包含文字，应控制文字数量，避免字号过小影响阅读。

该比例为宽高比 2:1，宽度为高度的两倍，呈现超宽的横向视野，信息承载量大。

默认实现通过 CSS `aspect-ratio: 2 / 1` 控制，兼容旧浏览器时使用 `padding-bottom: 50%`。

适用于 窄 Banner、运营横幅、列表顶部宣传图、模块引导图、轻量品牌露出图、跨栏宣传图等需要大面积横向展示的场景。

建议：2:1 适合用于轻量引导，不建议作为主要内容详情图。如果需要展示较完整的产品、人物或场景，应使用 16:9、3:2 或 4:3。

---

## 实现方式

### 推荐方案（现代浏览器）

优先使用 CSS `aspect-ratio` 属性，代码简洁、语义明确，浏览器自动根据宽度计算高度。

```css
.ratio-1-1   { aspect-ratio: 1 / 1; }
.ratio-4-3   { aspect-ratio: 4 / 3; }
.ratio-3-2   { aspect-ratio: 3 / 2; }
.ratio-16-9  { aspect-ratio: 16 / 9; }
.ratio-2-1   { aspect-ratio: 2 / 1; }
```

### 兼容方案（旧浏览器）

使用 `padding-bottom` 百分比模拟固定比例，容器需设置为相对定位，内部内容使用绝对定位。

```css
.ratio-1-1   { padding-bottom: 100%; }
.ratio-4-3   { padding-bottom: 75%; }
.ratio-3-2   { padding-bottom: 66.6667%; }
.ratio-16-9  { padding-bottom: 56.25%; }
.ratio-2-1   { padding-bottom: 50%; }
```

## 使用场景

**适用场景：**
- 页面中需要固定裁切比例的图片容器
- 卡片封面、视频封面、运营 banner 等视觉素材展示
- 头像、图标、宫格等需要规整正方形区域的内容

**不适用场景：**
- 无需固定比例的原始图片展示（使用自然宽高比）
- 需要自适应内容高度的容器（使用内容撑开方式）

---

## 注意事项

1. 同一页面内应保持图片比例统一，避免混用不同比例导致视觉不协调。
2. 比例越宽（如 2:1），信息承载量越大，但纵向空间占用相对较少，适合横向展示场景。
3. 比例越接近正方形（如 1:1），内容聚焦感越强，适合需要用户视线集中的区域。
4. 推荐优先使用 CSS `aspect-ratio` 实现，现代浏览器已广泛支持。
5. 兼容方案需注意容器定位方式，子内容需使用绝对定位避免撑开容器。
6. 图片素材裁切时需考虑比例对应的视觉中心区域，避免裁切后主体内容缺失。
7. 不同层级图片可使用不同比例，但需要保持清晰的视觉规律。
8. 图片比例应由容器控制，不应依赖图片原始尺寸自然撑开页面。
