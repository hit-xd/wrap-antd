# Pagination

**中文名称：** 分页
**英文名称：** Pagination
**所属分类：** 导航组件
**一句话简介：** 采用分页的形式分隔长列表。

---

## 文档说明

分页组件用于在数据量较大、当前页无法一次性展示全部数据时，对列表、表格、卡片流等内容进行分页浏览。

本规范包含 7 种分页形态：

1. 常用分页器：基础页码型
2. 常用分页器：页码 + 每页条数 + 跳转页
3. 常用分页器：总数 + 页码 + 每页条数 + 跳转页
4. 迷你分页器：基础页码型
5. 迷你分页器：页码 + 每页条数 + 跳转页
6. 迷你分页器：总数 + 页码 + 每页条数 + 跳转页
7. 简洁分页器：当前页输入 + 总页数展示型

---

# 一、常用分页器：基础页码型

## 1. 组件定义

常用分页器基础页码型由上一页按钮、页码按钮组、下一页按钮组成，用于最基础的数据翻页场景。

## 2. 使用场景

适用于只需要展示页码和前后翻页，不需要总记录数、每页条数和跳转页的列表或表格底部。

## 3. 结构组成

该形态包含：

1. 上一页按钮
2. 页码按钮组，包含页码：`1、2、3、4、5、6、7、8、9`。
3. 下一页按钮

## 4. 状态规则

### 默认状态

默认页码背景色：`#FFFFFF`  
默认页码边框颜色：`#000000   25%`  
默认页码文字颜色：`#000000   65%`  
默认页码文字字号：`14px`  
默认页码文字字重：`Regular`  
默认页码文字行高：`22px`

### hover 状态

hover 页码背景色：`#FFFFFF`  
hover 页码边框颜色：`#9B7235`  
hover 页码文字颜色：`#9B7235`  
hover 上一页 / 下一页图标颜色：`#9B7235`  
hover 上一页 / 下一页外框颜色：`#000000   25%`

### 当前页状态

当前页按钮背景色：`#FFFFFF`  
当前页按钮边框颜色：`#9B7235`  
当前页按钮边框宽度：`1px`  
当前页文字颜色：`#9B7235`  
当前页文字字号：`14px`  
当前页文字字重：`Regular`  
当前页文字行高：`22px`

### 禁用状态

上一页 / 下一页禁用状态边框颜色：`#000000   15%`  
上一页 / 下一页禁用状态图标颜色：`#000000   15%`  
禁用状态背景色：`#FFFFFF`  
禁用状态不响应 hover。

## 5. 尺寸 / 间距 / 颜色 / 交互规则

### 整体样式

整体分页高度：`32px`  
整体分页宽度：根据页码数量自适应，当前示例宽度为 `432px`  
整体背景：无填充、无描边  
分页排列方向：横向  
分页项之间的水平间距：`8px`

### 页码按钮样式

页码按钮尺寸：`32px × 32px`  
页码按钮圆角：`2px`  
页码按钮背景色：`#FFFFFF`  
页码按钮边框颜色：#000000   25%
页码按钮边框宽度：`1px`  
页码文字颜色：#000000   65%
页码文字字号：`14px`  
页码文字字重：`Regular`  
页码文字行高：`22px`  
页码文字字间距：`0px`  
页码文字与按钮边框的垂直间距：`5px`  
页码文字与按钮边框的水平间距：`12px`

### 上一页 / 下一页按钮样式

上一页 / 下一页按钮尺寸：`32px × 32px`  
上一页 / 下一页按钮圆角：`2px`  
上一页 / 下一页按钮背景色：`#FFFFFF`  
上一页 / 下一页按钮边框颜色：#000000   25%
上一页 / 下一页按钮边框宽度：`1px`  
上一页 / 下一页图标尺寸：`16px × 16px`  
上一页 / 下一页图标颜色：`#000000   65%`  
图标与按钮外边框的垂直间距：`8px`  
图标与按钮外边框的水平间距：`8px`

## 6. 注意事项

- 基于 Ant Design `Pagination` 封装时，应覆盖 `.ant-pagination-item`、`.ant-pagination-prev`、`.ant-pagination-next`、`.ant-pagination-item-active` 的默认样式。
- 当前页不使用蓝色边框和蓝色文字，统一改为 `#9B7235`。
- 页码按钮尺寸固定为 `32px × 32px`，不要使用 Ant Design 默认高度以外的随机尺寸。
