# Comment

**中文名称：** 评论
**英文名称：** Comment
**所属分类：** 数据展示
**一句话简介：** 评论组件，展示用户评论和沟通记录。

---

## 组件用法

**核心功能：** 评论组件用于展示用户评论或沟通记录，包含头像、作者、时间、内容和操作区。

**基础用法示例：**
```tsx
<Comment
  author="仲维鲁"
  avatar={<Avatar>仲</Avatar>}
  content={<p>客户资料已核实，建议安排面谈。</p>}
  datetime="2026-05-12 09:32"
/>
```

> 注意：该组件在 antd v5 中已标记为废弃。建议使用 Avatar + Typography 自行组合评论/沟通记录样式，或使用 List 布局替代。


## 字体说明

- 正文说明、标签、操作按钮和常规辅助信息默认使用 14px。
- 核心数字、统计值和标题按组件语义单独定义字号，不与正文混用。

---

## 使用场景

**适用场景：**
- 客户沟通记录展示
- 工作流审批意见

---

## API 文档

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| author | ReactNode | — | 作者 |
| avatar | ReactNode | — | 头像 |
| content | ReactNode | — | 评论内容 |
| datetime | ReactNode | — | 时间信息 |
| actions | ReactNode[] | — | 操作区 |

### Events

该组件为纯展示型组件，无事件回调。
