# Breadcrumb

**中文名称：** 面包屑
**英文名称：** Breadcrumb
**所属分类：** 导航组件
**一句话简介：** 显示当前页面在系统层级结构中的当前位置。

---

> 适用范围：本规范用于调整私行业务组件库中的「面包屑」组件样式。组件库基于 Ant Design 搭建，可基于 `Breadcrumb` 进行二次封装。  

---

## 1. 组件定义

面包屑组件用于展示当前页面在系统层级结构中的位置，并支持用户向上返回上一级或更高层级页面。

该组件属于导航辅助组件，不承担页面主导航功能，也不替代顶部导航、左侧导航或标签页。

组件基于 Ant Design `Breadcrumb` 二次封装时，应统一覆盖默认字体、颜色、间距和分隔符样式，避免出现 Ant Design 默认蓝色链接或默认间距与私行规范不一致的问题。

---

## 2. 使用场景

### 2.1 适用场景

用于需要表达页面层级关系较深的业务页面，三级及以上的页面才需要用到面包屑，或者需要返回两次才能返回到一级页面的情况下才需要用到面包屑。例如：

- 多级菜单进入的结果页
- 从列表页进入的子页面
- 需要支持返回上级页面的深层页面

典型结构示例：

```
首页 / 首页 / 当前页面
```

### 2.2 不适用场景

以下场景不建议使用面包屑：

- 一级首页或工作台首页
- 页面层级很浅、用户不需要回溯路径的页面
- 已经通过顶部导航、左侧导航、标签页明确表达当前位置的场景
- 弹窗、抽屉等局部容器中，除非有明确的层级回溯需求

---

## 3. 结构组成

面包屑组件由以下部分组成：

1. 外层面包屑容器
2. 普通层级项
3. 当前页面项
4. 分隔符

### 3.1 外层面包屑容器

外层容器用于承载所有层级项和分隔符。

整体高度：`22px`  
整体宽度：自适应  
背景：无填充  
描边：无  
布局方向：横向排列  
垂直对齐：居中对齐

### 3.2 普通层级项

普通层级项用于表示当前页面之前的上级页面。

普通层级项一般可点击，用于返回对应层级页面。

普通层级项文字内容示例：`首页`

### 3.3 当前页面项

当前页面项用于表示用户当前所在页面。

当前页面项通常不可点击，仅用于标识当前位置。

当前页面项文字内容示例：`当前页面`

### 3.4 分隔符

分隔符用于分隔不同层级项。

分隔符样式：`/`  
分隔符颜色：`#000000 45%`
分隔符与前后元素的间距：`8px`

---

## 4. 状态规则

### 4.1 普通页面状态

普通页面状态用于当前页面之前的层级项。

字体颜色：`#000000 45%` 
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`


hover 状态
hover 状态文字颜色：`#000000 85%`  
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`

### 4.2 当前页面状态

当前页面状态用于最后一级路径，表示当前所在页面。

字体颜色：`#000000 85%` 
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`

当前页面项通常不可点击。


### 4.3 分隔符状态

分隔符用于区分不同层级。

分隔符内容：`/`  
分隔符颜色：`#000000 45%` / `rgba(0, 0, 0, 0.45)`  
分隔符字号：跟随面包屑文字字号，`14px`  
分隔符行高：跟随面包屑整体行高，`22px`  
分隔符左右间距：`8px`
---

## 5. 尺寸 / 间距 / 颜色 / 交互规则

### 5.1 尺寸规则

整体高度：`22px`  
整体宽度：自适应  
单个文本项高度：`22px`  
文字字号：`14px`  
文字行高：`22px`  
字间距：`0px`

### 5.2 间距规则

元素间距：统一为 `8px`  
普通层级项与分隔符之间的间距：`8px`  
分隔符与后一个层级项之间的间距：`8px`  
组件上下内边距：`0px`，或由所在页面布局控制  
组件左右内边距：`0px`，或由所在页面布局控制

### 5.3 颜色规则

普通层级项文字颜色：`#000000 45%`
当前页面项文字颜色：`#000000 85%`
分隔符颜色：`#000000 45%`
背景色：透明 / 无填充  
描边颜色：无

### 5.4 交互规则

普通层级项点击行为：跳转至对应层级页面  
当前页面项点击行为：不可点击  
分隔符点击行为：不可点击
