# 栅格与间距

**中文名称：** 栅格与间距
**英文名称：** Space
**所属分类：** 布局
**一句话简介：** 栅格定页面结构，间距控元素呼吸感，两者共同构成统一的页面布局体系。

---

## 何时使用

当需要对页面进行整体布局框架搭建和空白距离控制时使用。栅格系统用于定义页面的结构和对齐，间距规范用于控制元素之间的空隙和呼吸感，两者共同构成统一的页面布局体系。

栅格系统适用于页面级的多列切分、表单字段排列、卡片网格等需要对齐的场景。Space 组件适用于模块内元素之间的间距控制。

---

## 基础栅格

基础栅格展示 24 栅格的基本切分方式，用于页面内容区的多列布局。

栅格系统由行（Row）和列（Col）两部分组成。Row 定义栅格行容器，Col 定义栅格列单元。每一行被平均分为 24 份，通过 `span` 属性控制每列占据的份数。

例如 `span={8}` 表示列占 8 份（即 8/24 = 1/3），`span={12}` 表示列占 12 份（即 12/24 = 1/2）。

行与行之间的垂直间距和列与列之间的水平间距通过 Row 的 `gutter` 属性控制。

---

## 左右偏移

通过 Col 的 `offset` 属性实现列之间的留白和错位布局，不需要额外使用空列来占位。

`offset` 属性表示列向左侧偏移的份数。例如 `span={10} offset={4}` 表示该列占据 10 份宽度，并在左侧留出 4 份空白。

偏移量适用于表单字段的错位排列、信息展示中的留白设计等需要打破均匀切分的场景。

---

## 响应式列

支持通过断点属性适配不同屏幕尺寸，实现响应式布局。

| 断点 | 屏幕宽度 | 说明 |
|------|----------|------|
| xs | `<576px` | 超小屏幕 |
| sm | `≥576px` | 小屏幕 |
| md | `≥768px` | 中等屏幕 |
| lg | `≥992px` | 大屏幕 |
| xl | `≥1200px` | 超大屏幕 |
| xxl | `≥1600px` | 极宽屏幕 |

例如 `xs={24} md={12} lg={8}` 表示在小屏设备上占满整行，中等屏幕占一半，大屏幕占三分之一。

---

## 12 列栅格系统

采用 12 列标准栅格系统，列间距（水槽 Gutter）统一为 `20px`，适配 `1440px` 与 `1920px` 双画布尺寸。

容器宽度公式为：`N列 + (N-1) × 20px（水槽）`。

例如 4 列 6 格布局：每列宽度的计算公式为 `(容器宽度 − 3 × 20px) ÷ 4`。

12 列栅格系统支持多种标准分法：
- 4 列均分（每列 `span={6}`）
- 三列均分（每列 `span={8}`）
- 非对称分栏（如 `span={4} + span={8}`、`span={18} + span={6}`）

---

## 间距规范（Spacing）

间距系统以 `4px` 为固定基数，所有尺寸必须是 4 的倍数。核心间距分三类：

| 分类 | 说明 | 示例 |
|------|------|------|
| 内边距 Padding | 元素内部，内容与边框之间的空白 | `padding: 16px` |
| 外边距 Margin | 元素外部，独立模块之间的空白 | `margin: 20px` |
| 槽间距 Gutter | 栅格布局中列与列之间的固定间距 | `gutter: 20px` |

### 标准尺寸分级

| 尺寸 | 使用场景 |
|------|----------|
| `4px` | 图标 + 文字、极小元素间隔 |
| `8px` | 按钮内部、小型组件间隔 |
| `12px` | 表单控件、中型组件间隔 |
| `16px` | 常规模块、内容区间隔 |
| `20px` | 栅格列间距、卡片主间距 |
| `24px` | 大模块、页面分区间隔 |
| `32px` | 页面级大间距、头部与内容间隔 |

### 核心原则

1. **统一原则** — 同一层级的元素或模块，间距必须完全一致。
2. **基数原则** — 所有间距必须是 4 的倍数，无例外。
3. **层级原则** — 信息层级越高，间距越大。
4. **固定原则** — 栅格槽间距固定 `20px`，不随屏幕变化。

## Space 间距组件

Space 组件用于在多个子元素之间设置统一间距，支持横向和纵向两种排列方向。

---

### 基础横向间距

用于排列标签、按钮等水平排列的组件组。通过 `size` 属性设置元素之间的统一间距。

支持三种预设间距值：
- `small` — `8px`，适用于紧凑排列
- `middle` — `16px`，适用于常规间距
- `large` — `24px`，适用于宽松排列

也支持直接传入数值自定义间距，例如 `size={12}`。

`wrap` 属性控制当容器宽度不足时子元素是否自动换行，避免内容溢出。

---

### 纵向排列

适用于信息分组堆叠，通过 `direction="vertical"` 实现垂直方向的统一间距。

纵向排列时，每个子元素占据独立一行，自上而下依次排列。间距规则与横向排列一致。

纵向排列适合操作按钮组、信息列表等需要垂直组织的场景。

---

### 分隔符间距

通过 `split` 属性在子元素之间快速插入分隔符，适用于操作分组、链接列表等需要视觉分隔的场景。

分隔符可使用 `Divider` 组件的垂直分割线类型（`type="vertical"`），在子元素之间插入细线分隔。

分隔符不占据间距空间，仅作为视觉分隔元素出现在子元素之间。

---

## 栅格 vs 间距

| | 栅格 Grid | 间距 Spacing |
|--|-----------|-------------|
| 定位 | 页面的整体布局框架 | 框架里的空白距离数值 |
| 作用 | 定规则、定结构、定对齐 | 定空隙、定呼吸、定层级 |
| 关系 | 槽间距（Gutter）既是栅格的一部分，也是间距的一种 |

没有栅格，间距会乱；没有间距，栅格会挤。两者相辅相成，共同构成页面布局规范。

## 设计规范

### 栅格 Token

| Token | 数值 | 说明 |
|-------|------|------|
| 栅格列数 | `12` | 标准栅格系统 |
| 水槽间距 | `20px` | 列间固定间距 |
| 画布标准 | `1440px` / `1920px` | 双画布尺寸 |

### Space Token

| Token | 数值 | 说明 |
|-------|------|------|
| gapSmall | `8px` | 小间距 |
| gapMiddle | `16px` | 中间距 |
| gapLarge | `24px` | 大间距 |

### API

**Space Props：**

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| size | 'small' \| 'middle' \| 'large' \| number | 'small' | 间距大小 |
| direction | 'horizontal' \| 'vertical' | 'horizontal' | 排列方向 |
| align | 'start' \| 'end' \| 'center' \| 'baseline' | 'center' | 对齐方式 |
| wrap | boolean | false | 是否自动换行 |
| split | ReactNode | — | 插入分隔符 |

**Row / Col Props：**

| 属性 | 组件 | 类型 | 说明 |
|------|------|------|------|
| gutter | Row | number \| [number, number] | 列间距（水平，垂直） |
| span | Col | number | 栅格占位格数（共 12 列） |
| offset | Col | number | 列左侧偏移格数 |
| xs / sm / md / lg / xl / xxl | Col | number \| object | 响应式断点配置 |

---

## 使用场景

**适用场景：**
- 页面内容区的多列栅格布局
- 表单字段的排列与对齐
- 模块内元素的间距控制
- 操作按钮组、标签组的间距

**不适用场景：**
- 页面的整体骨架搭建（请使用 Layout 布局组件）
- 弹性布局的复杂排列（请使用 Flex 弹性布局组件）

---

## 注意事项

1. 栅格系统使用 12 列而非 24 列，列间距统一为 `20px`。
2. 间距必须以 `4px` 为基数，所有间距值必须是 4 的倍数。
3. 同一层级的元素间距必须一致，避免视觉混乱。
4. 栅格槽间距固定为 `20px`，不随屏幕尺寸变化。
5. Space 组件默认水平排列，纵向排列需显式设置 `direction="vertical"`。
6. 当 Space 容器宽度不足时，建议开启 `wrap` 属性使子元素自动换行。
